# To Do List

## Tecnologias utilizadas
- React
- Tailwind

## Instruções para uso local

1. Digite um nome de usuário
2. Digite a tarefa a ser adicionada
3. Para marcar como concluída, clique em cima da tarefa
4. Para remover alguma tarefa da lista, clique no botão de remover
5. Para mudar de usuário, atualize a página e recomece do item 1
